37  * main(args) handles the processing of arguments as sequential actions to pass to the underlying XPathTool.
38  *
39  * @param args commandline arguments for processing:
40  * <ul>
41  * <li>-i loads a file (see {@link XPathTool#load(String)} )</li>
42  * <li>-o saves a file (see {@link XPathTool#save(String)} )</li>
43  * <li>-r sets a replacement value: anything "found" while running a "-f" argument is replaced by this value</li>
44  * <li>-f starts a search, replacing the value of all matching nodes and attributes with the replacement (-r) value given</li>
45  * <li>-R sets a replacement pattern: ie /from/to/g : anything "found" while running a "-f" argument, all "from" are replaced by "to". "g" as the last parameter allows multiple replacements per matching node</li>
46  * <li>-F starts a search, triggering -R replacement on values on all matching nodes</li>
47  * <li>-V shows version and quits</li>
48  * </ul>
49  */
