Commandline Options
Global XPathSet.main (String[] args)

-F execute a search using the replacement already registered: for each matching node, execute the replacement

java -jar xpathtool.jar -i .. -r .. -F "//ScanTask[@name='VWVersionSearch']/@file" -o ..

The format of the search string is a standard XPath expression

See also
http://en.wikipedia.org/wiki/XPath
http://www.w3.org/TR/xpath/

-o save an XML file after editing

-i read in an XML file for editing

-H Show a simple help screen as a reminder of options which are understood by the application

java -jar xpathtool.jar --help

-V Show the current release version for reference

java -jar xpathtool.jar -V
